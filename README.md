# pengondisian

penjelasan dan fungsi

Git Init
Perintah untuk membuat (menginisialisasi) repository Git baru di folder project kamu.

Git status
Menampilkan kondisi terbaru dari project Git kamu.

Git add
digunakan untuk memasukkan perubahan file ke staging area sebelum disimpan (commit).

Git commit
Menyimpan perubahan yang sudah di-add ke dalam repository Git.

Git push origin
Mengirim (upload) commit dari komputer lokal ke repository online (misalnya GitHub).
